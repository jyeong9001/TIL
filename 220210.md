# Space bar click시 html page가 scrolling되는 것을 방지하는 방법 (stackoverflow)

```js
window.onkeydown = function (e) {
  return !(e.keyCode == 32);
};
```

# target이 body일 경우 해결 방법

```js
window.addEventListener("keydown", function (e) {
  if (e.keyCode == 32 && e.target == document.body) {
    e.preventDefault();
  }
});
```
